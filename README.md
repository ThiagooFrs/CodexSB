# CodexSB

Este repositório contém um exemplo de script Python que funciona como relógio com alarme.

## Como usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o script:
   ```bash
   python3 clock_alarm.py
   ```
   Escolha entre o modo relógio com alarme ou o modo pomodoro.
   Para que o alarme funcione, coloque um arquivo `alarme.mp3` na mesma pasta do script.
